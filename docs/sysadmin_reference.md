## HPC Tailoring Hotspots
- **audit-volume-compute-nodes** (3.3.1): Per-process logging at scale can overwhelm storage and analysis systems.
- **container-security** (3.4.8): Researchers need portable containers that may include privileged runtime options.
- **fips-cryptography-infiniband** (3.13.8): Native InfiniBand encryption choices can reduce bandwidth and increase latency.
- **gpu-memory-sanitization** (3.8.3): [GPU](glossary_full.md#term-gpu) memory can retain residual data across workload boundaries.
- **long-running-job-management** (3.13.12): Scientific jobs may require uninterrupted runtimes over extended windows.
- **mfa-batch-jobs** (3.5.3): Non-interactive scheduler jobs cannot satisfy prompt-based [MFA](glossary_full.md#term-mfa) at execution time.
- **multi-tenancy-isolation** (3.13.2): Shared nodes and accelerators increase tenant co-residency risk.
- **node-provisioning-pxe** (3.4.1): Rapid node reprovisioning relies on network boot workflows vulnerable to tampering.
- **parallel-filesystem-acls** (3.1.3): High-throughput filesystems may not support enterprise [ACL](glossary_full.md#term-acl) semantics uniformly.
- **session-timeout-compute-nodes** (3.13.11): Long-running batch jobs often execute for days without interactive activity.
